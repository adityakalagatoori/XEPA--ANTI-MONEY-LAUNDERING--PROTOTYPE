# 🔍 XEPA — Anti-Money Laundering (AML) Detection & Monitoring System

### Predictive Financial Crime Intelligence Platform

XEPA is a next-generation Anti-Money Laundering (AML) monitoring and investigation platform designed to simulate enterprise-grade compliance infrastructure with predictive intelligence, cryptographic audit verification, and investigator-focused workflows.

Built as a high-fidelity MVP, XEPA combines:
- Predictive transaction analysis
- Risk intelligence systems
- Graph-based financial tracing
- Cryptographic audit trails
- AI-assisted compliance investigation

The platform delivers a premium end-to-end workflow for identifying, analyzing, and investigating suspicious financial activities.

# 📌 About The Project

Modern financial systems process millions of transactions daily, making manual monitoring nearly impossible.

Traditional AML systems often suffer from:
- Static rule limitations
- High false-positive rates
- Weak predictive capabilities
- Fragmented investigation workflows
- Lack of tamper-proof audit systems

XEPA addresses these gaps using predictive intelligence and cryptographic verification to create a smarter, investigator-centric AML ecosystem.

The platform is designed to:
- Detect suspicious financial behavior
- Predict laundering patterns
- Visualize transaction relationships
- Streamline investigator workflows
- Maintain immutable compliance audit trails

Money laundering operations have evolved into highly sophisticated financial networks involving:

# ⚙️ Workflow

1. Transactions enter the monitoring system
2. Behavioral analytics evaluate transaction patterns
3. Risk scoring engine calculates anomaly levels
4. Graph intelligence maps entity relationships
5. Ghost-node forecasting predicts potential laundering paths
6. Suspicious activities trigger alerts
7. Investigators review flagged entities
8. AI generates investigation narratives
9. Every action is securely hashed into the audit ledger

# 🚀 Run Using Automated Scripts

The project includes PowerShell automation scripts for simplified local setup.

---

## 1️⃣ Start Backend

Open PowerShell and run:

```powershell
./scripts/run-backend.ps1
```

This will:
- Create/check virtual environment
- Install backend dependencies
- Start FastAPI server

Backend runs at:

```text
http://127.0.0.1:8000
```

---

## 2️⃣ Start Frontend

Open another PowerShell terminal and run:

```powershell
./scripts/run-frontend.ps1
```

This will:
- Install frontend packages
- Launch Vite development server

Frontend runs at:

```text
http://127.0.0.1:5173
```

---

# 🔧 Manual Setup

# Backend Setup

Navigate to backend directory:

```bash
cd backend
```

Create virtual environment:

```bash
python -m venv .venv
```

Activate environment:

### Windows

```powershell
.\.venv\Scripts\Activate.ps1
```

### macOS/Linux

```bash
source .venv/bin/activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run FastAPI server:

```bash
python -m uvicorn app.main:app --reload --host 127.0.0.1 --port 8000
```

---

# Frontend Setup

Navigate to frontend:

```bash
cd frontend
```

Copy environment variables:

```bash
copy .env.example .env
```

Install dependencies:

```bash
npm install
```

Run frontend server:

```bash
npm run dev -- --host 127.0.0.1 --port 5173
```
